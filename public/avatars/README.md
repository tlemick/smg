## Image Requirements:
- Format: JPG or PNG
- Size: Recommended 200x200px or higher (will be scaled down)
- Square aspect ratio works best
- Professional headshots or realistic portraits

## How it Works:
The avatar service uses a hash of the user ID to consistently assign the same avatar to each user, but the assignment appears random across different users.

## For Production:
Replace this system with actual user-uploaded profile images stored in a cloud service like AWS S3, Cloudinary, etc.
